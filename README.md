# Concat-streams-in-java
Stream.concat() is a static method in Java Streams API used to merge two streams into a single stream.

The program rearranges numbers:
  - Negative  numbers first
  - Positive numbers next
  - Zeros atthe end

    #Example Input
    (-1,0,7,0,-3,8,5,0)
    # Output
    (-1,-3,7,8,5,0,0,0)
